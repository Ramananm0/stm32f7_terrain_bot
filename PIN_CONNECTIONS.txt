══════════════════════════════════════════════════════════════════════
  1. BTS7960 FL — Front Left Motor
══════════════════════════════════════════════════════════════════════
  BTS7960    Arduino    STM32    Timer         Note
  RPWM    →  A4         PF7      TIM11_CH1     forward PWM
  LPWM    →  A5         PF6      TIM10_CH1     reverse PWM
  R_EN    →  3.3V       —        —             tie HIGH always
  L_EN    →  3.3V       —        —             tie HIGH always
  VCC     →  5V         —        —
  GND     →  GND        —        —
  B+ / B- →  FL motor terminals

══════════════════════════════════════════════════════════════════════
  2. BTS7960 FR — Front Right Motor
══════════════════════════════════════════════════════════════════════
  RPWM    →  A2         PF9      TIM14_CH1     forward PWM
  LPWM    →  A3         PF8      TIM13_CH1     reverse PWM
  R_EN    →  3.3V       —        —
  L_EN    →  3.3V       —        —
  VCC     →  5V         —        —
  GND     →  GND        —        —
  B+ / B- →  FR motor terminals

══════════════════════════════════════════════════════════════════════

══════════════════════════════════════════════════════════════════════

  L_EN    →  3.3V       —        —
  VCC     →  5V         —        —
  GND     →  GND        —        —

  L_EN    →  3.3V       —        —
  VCC     →  5V         —        —
  GND     →  GND        —        —

══════════════════════════════════════════════════════════════════════
  5. ICM-20948 IMU → CN2 connector (8-pin I2C extension)
══════════════════════════════════════════════════════════════════════
  ICM-20948    CN2 Pin    STM32
  VDD       →  Pin 5      +3V3
  GND       →  Pin 7      GND
  SDA       →  Pin 1      PB9 (I2C1_SDA)
  SCL       →  Pin 3      PB8 (I2C1_SCL)
  AD0       →  GND        addr = 0x68
  INT       →  not connected

══════════════════════════════════════════════════════════════════════
  6. ESP32-WROOM encoder co-processor → I2C1 shared bus
══════════════════════════════════════════════════════════════════════
  ESP32        Arduino    STM32
  GPIO21(SDA)→  D14        PB9 (shared with IMU)
  GPIO22(SCL)→  D15        PB8 (shared with IMU)
  3V3        →  3V3 pin    STM32 3.3V output
  GND        →  GND        common ground

  I2C address: 0x30

  Encoder connections to ESP32:
  Motor    CH_A (green)    CH_B (white)
  FL    →  GPIO4           GPIO5
  FR    →  GPIO12          GPIO13
  RL    →  GPIO14          GPIO15
  RR    →  GPIO33          GPIO32

  Motor encoder power:
  Red  (5V)  → 5V rail
  Black(GND) → GND

══════════════════════════════════════════════════════════════════════
  7. I2C pull-up resistors
══════════════════════════════════════════════════════════════════════
  4.7kΩ between SDA (D14/PB9) and 3.3V
  4.7kΩ between SCL (D15/PB8) and 3.3V

  Note: check if ICM-20948 breakout board already has pull-ups.
  If yes, external resistors not needed.

══════════════════════════════════════════════════════════════════════
  8. UART to Raspberry Pi (USB-TTL cable)
══════════════════════════════════════════════════════════════════════
  Cable wire    Arduino    STM32
  White(RXD) →  D1         PC6   USART6_TX
  Green(TXD) →  D0         PC7   USART6_RX
  Black(GND) →  GND        GND
  Red        →  DO NOT CONNECT
  Yellow/Blue→  DO NOT CONNECT

  Baud rate: 2,000,000

══════════════════════════════════════════════════════════════════════
  9. Complete Arduino header pin usage summary
══════════════════════════════════════════════════════════════════════
  Label  STM32   Function                    Timer
  A2     PF9     RR RPWM (forward)           TIM14_CH1
  A3     PF8     FR LPWM (reverse)           TIM13_CH1  ← note: FR reverse
  A4     PF7     FL RPWM (forward)           TIM11_CH1
  A5     PF6     FL LPWM (reverse)           TIM10_CH1
  D0     PC7     UART RX ← RPi              USART6_RX
  D1     PC6     UART TX → RPi              USART6_TX
  D3     PB4     RR LPWM (reverse)           TIM3_CH1
  D6     PH6     RL RPWM (forward)           TIM12_CH1
  D10    PA8     RR RPWM (forward)           TIM1_CH1
  D11    PB15    RL LPWM (reverse)           TIM12_CH2
  D14    PB9     I2C SDA (IMU + ESP32)       I2C1_SDA
  D15    PB8     I2C SCL (IMU + ESP32)       I2C1_SCL

  Free pins: A0, A1, D2, D4, D5, D7, D8, D9, D12, D13

══════════════════════════════════════════════════════════════════════
  10. CubeMX peripheral configuration
══════════════════════════════════════════════════════════════════════
  Peripheral  Pins              Settings
  I2C1        PB8, PB9          Fast Mode 400kHz
  USART6      PC6, PC7          2000000 baud 8N1 no flow ctrl
  TIM10       PF6 (A5)          PWM CH1 PSC=0 ARR=10799 Mode1 High
  TIM11       PF7 (A4)          PWM CH1 PSC=0 ARR=10799 Mode1 High
  TIM12       PH6+PB15 (D6+D11) PWM CH1+CH2 PSC=0 ARR=10799 Mode1 High
  TIM13       PF8 (A3)          PWM CH1 PSC=0 ARR=10799 Mode1 High
  TIM14       PF9 (A2)          PWM CH1 PSC=0 ARR=10799 Mode1 High
  TIM1        PA8 (D10)         PWM CH1 PSC=0 ARR=10799 Mode1 High
  TIM3        PB4 (D3)          PWM CH1 PSC=0 ARR=10799 Mode1 High

  Clock: HSE bypass 25MHz → PLLM=25 PLLN=432 PLLP=2 → 216MHz

══════════════════════════════════════════════════════════════════════
  11. Breadboard wiring for I2C bus
══════════════════════════════════════════════════════════════════════
  Use a small breadboard to split the I2C bus:

  Row 1 (SCL): STM32 D15 → ICM SCL → ESP32 GPIO22 → 4.7kΩ → 3V3
  Row 2 (SDA): STM32 D14 → ICM SDA → ESP32 GPIO21 → 4.7kΩ → 3V3
  Row 3 (3V3): STM32 3V3 → ICM VDD → ESP32 3V3
  Row 4 (GND): STM32 GND → ICM GND → ESP32 GND → all motors GND
